## Desenvolvimento

### Instalar e executar:

```bash
yarn install
yarn start
```

ou

```bash
npm install
npm start
```

### Acessar aplicação

Depois de iniciado, aplicação estará rodando na porta *3000*:

> http://localhost:3000

## O que contém no boilerplate

- Razzle
- Redux
- React
- React Router v5.0
- Less
